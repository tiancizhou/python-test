# Python 三个魔术方法示例设计

**目标**：在 `类和对象` 目录中新增一个可直接运行的示例文件，演示 `__init__`、`__str__`、`__repr__` 的作用。

**范围**：
- 新增一个 `Person` 类示例
- 提供主程序运行输出
- 增加最小测试，验证三个魔术方法的行为

**设计选择**：
- 使用单文件示例，降低入门理解成本
- 使用 `unittest`，避免引入额外依赖
- 测试覆盖对象初始化、字符串显示和调试表示

**实现结果预期**：
- 运行示例文件时，能直观看到 `print(obj)` 与 `repr(obj)` 的区别
- 测试可验证 `Person` 类的三个魔术方法行为
